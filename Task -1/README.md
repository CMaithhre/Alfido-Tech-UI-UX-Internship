**Alfido Tech UI/UX Internship**

_Task 1 - Mobile App UX Case Study_

Project Overview:

Designing a mobile application for freelancers to centrally manage projects, track earnings, monitor clients, and handle billing.

Problem Statement:

Freelancers often struggle with fragmented workflows, using multiple disconnected tools to manage daily operations like deadlines, financial metrics, and client invoicing.
